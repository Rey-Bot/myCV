Projects

  1. Traffic Management using Smart Lane Divider System

    This system is a smart movable divider which is used to manage and reduce traffic within the city limits.

    • Image Processing technique with Viola-Jones Algorithm is used to determine traffic intensity in both

      lanes using Matlab. After determining the most crowded lane, which is the output of image

      processing, it decides which side the divider has to move. That data/signal is sent to the divider.

    • NodeMCU and L293D Motor drivers are used for moving the divider.


  2. Quadcopter

    Built a quadcopter using a Thunder QQ Board

    • Thunder QQ Board, a flight control system is first calibrated on all 3 axis. Four brushless

      dc motors are connected to the board which is attached to the frame. Power Distribution Board

      equally distributes power to all the components in quadcopter. Receiver is fixed to the board to

      receive signal from the transmitter.

    • A 2.4Ghz frequency, 1km range Transmitter/Remote is used to control the quadcopter.
